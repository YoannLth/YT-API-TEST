Youtube Channel Videos

F.A.Q

Utilisation

Rentrer un nom d'utilisateur YouTube dans la barre de recherche en haut à droite

Technologies utilisés

HTML / JQuery / AJAX / YouTube API

Comment ça fonctionne?

Lors d'un clic sur le bouton de recherche (icone loupe), le texte du champ de texte est récupéré. Une requête AJAX est émise construite pour récupérer l'ID de la chaîne YouTube qui correspond à l'utilisateur recherché.

Cette requête est envoyé sur l'API officielle de YouTube ("https://www.googleapis.com/youtube/v3/..."). La requête renvoi un fichier JSON avec les informations demandés et l'application crée une deuxième requête pour récupérer les 10 dernières vidéos

Enfin, une fonction JQuery récupère l'ID des vidéos et crée un iFrame pour chaque/
